# EventFlow - Projekthanteringsapp

En komplett projekthanteringsapp för eventbyråer med:
- ✅ Inloggning & användarkonton
- ✅ Synkning mellan enheter
- ✅ Projekt, uppgifter, risker
- ✅ Bilduppladdning & komprimering
- ✅ Anteckningar med länkar & filer
- ✅ Arbetsorder
- ✅ Varumärkesfärger

## Teknisk stack
- React + Vite
- Supabase (databas & auth)
- Tailwind CSS
- Vercel (hosting)
